Past Projects

ChessMate

Electron · React · Vite · Node.js · Native Addons

ChessMate is a cross-platform desktop chess application built with Electron, Vite, and React, designed for local play and learning. It integrates a custom Node.js native addon to power move validation, analysis, and engine communication—allowing players to practice, analyze games, and play without relying on online services.

Schedule-1-Helper

Electron · React · Vite · Node.js · Save File Parsing · Data Modeling

Schedule-1 Helper is a cross-platform desktop companion app built with Electron, Vite, and React that reads and interprets local game save data from Schedule 1. It automatically locates the user's save file, extracts discovered strain and recipe data, and presents it in a clean interface—allowing players to analyze discovered strains and experiment with new combinations using a built-in mixer tool.
